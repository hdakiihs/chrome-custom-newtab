# Custom New Tab

Chromeの新しいタブページをカスタマイズするChrome拡張機能です。

## 機能

- **時計表示** - 現在時刻と日付をリアルタイム表示
- **検索バー** - Enterキーで検索を実行
- **天気表示** - 現在地の天気と気温を表示（位置情報を使用）
- **月間カレンダー** - 日本の祝日に対応
- **Googleカレンダー連携** - 今日・明日の予定を表示
- **ショートカット** - よく使うサイトへのリンクを管理
- **テーマ切り替え** - ライト/ダーク/自動モード

## インストール

1. このリポジトリをクローン
   ```bash
   git clone https://github.com/hdakiihs/chrome-custom-newtab.git
   ```

2. Chromeで `chrome://extensions` を開く

3. 「デベロッパーモード」を有効にする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. `src` フォルダを選択

## 使い方

### ショートカット

- クリック: サイトを開く
- 右クリック または Ctrl/Cmd + クリック: 編集
- 「+」ボタン: 新しいショートカットを追加

### カレンダー

- 日付をクリック: その日の予定を表示
- 「今日に戻る」ボタン: 今日・明日の予定表示に戻る

## 必要な権限

- `search` - 検索機能
- `identity` - Googleカレンダー連携
- `storage` - 設定・キャッシュの保存
